
## 🎬 AI Movie Insight Builder

The AI Movie Insight Builder is a Next.js-based web app that fetches movie details using an IMDb ID and analyzes audience reviews using the Google Gemini API to generate an AI-powered sentiment summary. It combines data from OMDb and TMDB APIs to provide users with both factual movie information and an aggregated view of audience opinion in a clean, interactive interface.
## 🚀 Setup Instructions
🎯 Technology 


Tech Stack: Next.js, React, Tailwind CSS, Axios, Node.js, OMDb API, TMDB API, Google Gemini API






## Install the Dependencies
- npm install


## Create .env.local

- OMDB_API_KEY=your_omdb_key TMDB_API_KEY=your_tmdb_key GEMINI_API_KEY=your_gemini_key

## Run Server

- npm run dev =>Application is Running on http://localhost:3000
